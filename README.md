# Unisatc — Atividades de Clean Code

Repositório para organizar e versionar as atividades desenvolvidas durante as aulas de **Clean Code** na **UniSatc**, com o **professor Ramon**.

## Objetivo

- Centralizar as atividades e exercícios feitos em aula
- Manter um histórico de evolução (commits) ao longo do semestre
- Facilitar a revisão/estudo e a reutilização de exemplos

## Estrutura do repositório

A estrutura pode variar conforme as aulas, mas a organização sugerida é:

- `atividades/`
  - `aula-01/`
  - `aula-02/`
  - ...
- `projetos/` (quando houver)
- `anotacoes/` (resumos e observações)

> Se preferir outra organização, fique à vontade — o importante é manter consistência.

## Como executar

A maior parte do conteúdo está em **Jupyter Notebook**.

1. Instale o Python (recomendado 3.10+)
2. Crie um ambiente virtual (opcional, mas recomendado)
3. Instale o Jupyter

Exemplo:

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

pip install jupyter
jupyter notebook
```
