# Unisatc — Atividades de Clean Code

Repositório para organizar e versionar as atividades desenvolvidas durante as aulas de **Clean Code** na **UniSatc**, com o **professor Ramon**.

## Objetivo

- Centralizar as atividades e exercícios feitos em aula
- Manter um histórico de evolução (commits) ao longo do semestre
- Facilitar a revisão/estudo e a reutilização de exemplos
